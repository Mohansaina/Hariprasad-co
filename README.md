# Hariprasad & Co - Mobile UI Design

A high-fidelity mobile UI design for Hariprasad & Co, a professional tax firm specializing in tax and GST solutions in Shamshabad.

## Features

- Clean, modern, and trustworthy aesthetic
- Color palette: White, navy blue (#003366), and emerald green (#00A86B)
- Professional CA portrait at the top
- Clear headline: "Expert Tax & GST Solutions in Shamshabad"
- Prominent "Chat on WhatsApp" button
- 5.0 Stars (68 Reviews) rating badge
- Service icons for "ITR Filing", "GST Registration", and "Financial Planning"
- Responsive design optimized for mobile devices
- Minimalist design with professional lighting

## Technology Stack

- HTML5
- CSS3 (with Flexbox and Grid)
- JavaScript (ES6+)
- Font Awesome for icons
- Google Fonts (Poppins)

## How to Run

1. Open `index.html` in any modern web browser
2. For best experience, view on a mobile device or use browser's device emulator

## Customization

- To change the WhatsApp number, update the `href` in the WhatsApp button
- To update contact information, modify the contact section in `index.html`
- Colors can be adjusted in `css/style.css`
- Services can be modified in the services grid section
